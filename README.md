# INSTALLATION

1. Install dependencies
```
git clone https://github.com/Bardo91/hecatonquiros
cd hecatonquiros
bash scripts/install_dependencies.sh
```
2. Compile library and install
```
cd build
cmake ..
make -j4
sudo make install
```
